# Eletro Link Manutenções ⚡

Site institucional desenvolvido para cliente real na área de manutenções elétricas.

## 🔗 Deploy
[eletrolinkmanutencoes.com](https://eletrolinkmanutencoes.com)

## ✨ Funcionalidades
- Página institucional com serviços e contato
- Formulário de solicitação de orçamento
- Painel administrativo protegido por login
- Integração com Supabase (banco de dados e autenticação)

## 🛠️ Tecnologias
- React + TypeScript + Vite
- Supabase (Auth + Database)
- Vercel (deploy)
- Tailwind CSS
